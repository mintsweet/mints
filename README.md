## Mints

> simple and efficient h5 development tools.

## Features

  - Easy to use.
  - Zero config by default.
  - Eliminating webpack complex config.
  - Reduced access costs.

## Quick start

1. init project

```bash
npx mints new project_name
```

2. develop your page

```bash
$ mints start
```

3. build your project

```bash
$ mints build
```

# Popup 蒙层

### 使用指南

```javascript
import { Popup } from 'mints-ui';

this.popup = new Popup();
```

### 代码演示

#### 显示

```javascript
this.popup.show();
```

#### 隐藏

```javascript
this.popup.hide();
```

### 属性

| 参数 | 类型   | 默认值 | 说明     |
| :--- | :----- | :----- | :------- |
| body | String | ''     | 蒙层内容 |

### 方法

| 方法名 | 参数 | 说明 |
| :----- | :--- | :--- |
| show   | 无   | 显示 |
| hide   | 无   | 隐藏 |
